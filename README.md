# Análise Multimodal de Conteúdo Viral

Sistema de pesquisa focado na extração de padrões estruturais em conteúdos digitais virais.

O objetivo é compreender como estruturas visuais, narrativas e emocionais influenciam comportamento do usuário em escala.

Este projeto explora a interseção entre:

- análise de conteúdo viral
- engenharia emocional aplicada a marketing
- estruturas narrativas
- sistemas de growth
- inteligência artificial multimodal

---

# Ideia Central

Conteúdos virais raramente são aleatórios.

Conteúdos de alto desempenho costumam seguir padrões estruturais recorrentes em diferentes camadas:

- composição visual
- tensão narrativa
- gatilhos emocionais
- loops de curiosidade
- arquétipos simbólicos

Ao decompor conteúdos nessas camadas é possível identificar padrões replicáveis.

# Camadas de Análise

A análise multimodal de conteúdo viral é dividida em diferentes camadas estruturais.

Cada camada extrai um tipo específico de padrão responsável por capturar atenção e influenciar comportamento do usuário.

## Camada Visual

Analisa padrões de composição visual responsáveis por gerar curiosidade e impacto imediato.

Componentes analisados:

- composição cinematográfica
- hierarquia visual
- contraste entre personagem e fundo
- iluminação dramática
- símbolos narrativos

Essa camada é responsável por extrair o **DNA visual (VDNA)** de thumbnails e imagens virais.

---

## Camada Narrativa

Analisa estruturas narrativas utilizadas para gerar curiosidade e retenção.

Elementos analisados:

- promessa narrativa
- estrutura de mistério
- revelação implícita
- tensão dramática
- arquétipos narrativos

Essa camada busca identificar **padrões de storytelling recorrentes** em conteúdos virais.

---

## Camada Emocional

Analisa vetores emocionais presentes no conteúdo.

Componentes analisados:

- curiosidade
- ameaça
- autoridade
- revelação
- surpresa

Esses vetores influenciam diretamente o comportamento de clique e retenção.
